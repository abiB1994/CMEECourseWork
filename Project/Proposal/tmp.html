  \begin{abstract}
  Can Machine learning techniques correctly identify multiple stresses on a
    hyperspectral leaf image, and if so, to what accuracy and can it differentiate between
    the stresses and the responses?
   
    Keywords: Digital Pathology, Random Forests, Python, Disease detection,
    Multispectral features, Machine Learning.
  \end{abstract}
 
  \section{Introduction}
  Over the recent years, the field of digital pathology has exploded; searching
    digital pathology on Web of Science returns over 3,000 hits, and 2013 onwards
    returns over 250 papers per annum. Despite this, the avenue of digital pathology
    in plants is lagging behind, and so we believe it to be a good research avenue
    to go down to expand upon current knowledge.
   
    The study of plant pathology has always recieved much attention, due to it's link with
    agricultural yield and crop productivity \cite{donatelli_modelling_2017, waller_the_2005}.
    Despite most critical diseases of crops having been controlled \cite{wood_sustainable_1996}
  the diseases that still manage to impact upon crop production can be devastating
    \cite{woodham-smith_great_1962, mccook_global_2006}. Pathogens can affect the host in multiple
    ways, with signs of the disease showing on the leaves or on the fruit.

    Past methods in detecting plant disease rely on experts simply observing the plant
    \cite{singh_detection_2017}; despite the rise in the use of machine learning for classification,
    many still rely on manual detection in this field. Our aim is to create a functioning web application
    to allow for researchers to analyse their images using a Random forest approach on
    multispectral features of the image. We will use a partially supervised approach
    to allow for recognised plant disease experts to identify diseased leaves for a training
    set.
   
    The main aim of this project is to see whether image classifiers can correctly identify multispectral
    features from multiple stresses. We are also going to build a fully functional web application to allow
    users the chance to analyse their image data in an analyse section, and to have a contribute section, whereby
experts can further train the classifier with their identified features in a train/test format.We anticipate
that this project will be successful due to previous work done using machine learning and image analysis,
and that this project could be vastly influential to other areas utilysing image analysis techniques.
   
\section{Materials \& Methods}

The proposed methods are to gather multispectral image data from \textit{Arabidopsis thaliana} and
tomato plants, all of which would be infected with \textit{Botrytis cinerea}, and some of which are further
exposed to high levels of Ultra Violet (UV) light.

These images would then be classified on features within the objects (i.e. leasions on the leaf image,
not features in the background), and would be divided into a train and test set of unequal distribution.
We would then train our classifier on the train images, until it could sufficiently return high accuracy
on the test images.

Alongside this, we would also make the Web application.
